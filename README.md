# E-Commerce Business Analyzer

The **E-Commerce Business Analyzer** is a data analytics and machine learning project designed to analyze e-commerce business data and generate meaningful insights. The system helps businesses understand their sales performance, customer behavior, product demand, inventory status, and delivery operations.

The project uses **Python, machine learning models, and data visualization techniques** to process large datasets and transform raw data into useful business insights. The dashboard allows users to easily explore trends, predictions, and analytics results through interactive charts and graphs.

---

## Project Objectives

The main objective of this project is to analyze e-commerce data and generate useful insights that can help businesses make better decisions. The system focuses on important business areas such as:

* Sales performance analysis
* Customer behavior analysis
* Product demand and forecasting
* Inventory management
* Delivery and logistics performance

By analyzing these factors, the system helps businesses identify trends, patterns, and potential improvements.

---

## Key Features

* Product analytics and demand analysis
* Customer behavior analysis and segmentation
* Sales forecasting and revenue insights
* Inventory and delivery performance analysis
* Data visualization dashboard for easy understanding
* Machine learning models for predictions and insights

---

## Technologies Used

The project is developed using the following technologies:

* **Python** – Data processing and machine learning
* **Machine Learning Libraries** – Scikit-learn, Pandas, NumPy
* **Data Visualization** – Matplotlib, Seaborn
* **Streamlit** – Analytics dashboard
* **Flask API** – Backend API for data processing
* **Next.js** – Frontend web interface
* **Node.js** – Frontend development environment

## Project Structure

data/ → Raw datasets used for analysis

src/data_prep/ → Data cleaning and preprocessing

src/analytics/ → Business analytics modules
src/visualization/ → Dashboard and visualization code
frontend/ → Web frontend interface
artifacts/ → Generated outputs and plots

## How to Run the Project

### 1. Install dependencies

pip install -r requirements.txt

---

### 2. Run the main data pipeline

python main.py

---

### 3. Run the Streamlit analytics dashboard

streamlit run app.py

---

### 4. Run the Web Application

Start the backend API:

python flask_api.py

Open another terminal and start the frontend:

cd frontend
npm install
npm run dev

Then open the following link in your browser:

http://localhost:3000
